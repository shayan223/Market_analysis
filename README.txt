Data found at https://www.kaggle.com/prasoonkottarathil/ethereum-historical-dataset

download and extract the 3 data csv's into a folder called 'data'. the script should cover the rest.

To run:
1. Download and extract dataset to ./data directory
2. run generate_daily_plots.py to create generate requisite plots/graphs
3. run train_models.py to train approximators
4. run deep_q_agent.py to use those approximators to train agent
